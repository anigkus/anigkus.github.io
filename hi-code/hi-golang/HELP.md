# Generated go.mod file
```
go mod init github.com/anigkus/hi-golang
```  

# Installing dependencies
```
go get github.com/lib/pq
go get github.com/lib/pq@master
go get github.com/lib/pq@v1.8.0
```

# Removing unused dependencies
```
go mod tidy
```

# Build
```
go build main.go
```
